About Me

I started game development in 2006 on Game Maker 7. From then on, I've always been passionate about games and coding, doing freelancing and getting a diploma in Web Development and Design from NAIT.

Using technical and design skills from that, I started to seriously pursue game dev in 2023 under the name Pyrano Studios, making the large scale tactical RPG Prismatic: Nox's Gate and other games like Akansi and Taco Doomsday. I primarily use Godot and also do sound design, music, art and animation.

My Work

Prismatic: Nox's Gate

Full-length tactical RPG implementing turn-based combat system with large teams, voice acting, open world sections, and many abilities.

Primeval Age Web RPG

Implemented extensions and upgrades to large-scale web game Primeval Age using Laravel framework and Lorekeeper.

100% Audio Game

(in beta) Sounds Terrible is a game for the visually impaired or anyone looking for a unique experience. The name is a pun on the fact that the player is pursued by a terrifying creature that scrapes across the floor. It uses a custom positional audio system to represent the world.

Infernal Knight

I'm a part-time developer at Troupial Studios for their upcoming agility platformer. Implemented the player controller, state machine, environment, and custom platforming mechanics.

Taco Doomsday

Taco Doomsday is a Backrooms survival/nextbot game where the player investigates endless hallways behind a restaurant while being chased by various food-based monsters!

Tools and Websites
